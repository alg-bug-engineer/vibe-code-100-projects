## 🎯 下一步

### 建议操作顺序

1. **安装依赖**
   ```bash
   pip install gradio
   ```

2. **测试系统**
   ```bash
   python test_ui.py
   ```

3. **启动界面**
   ```bash
   python ui.py
   # 或
   ./start.sh  # 选择选项1
   ```

4. **配置系统**
   - 在浏览器中打开 http://localhost:7860
   - 进入"系统配置"标签页
   - 输入API Key并初始化

5. **开始使用**
   - 推荐使用"一键流程"快速体验
   - 或按需使用单独的功能标签页
